Documentação de Sombras

Esta página demonstra as classes utilitárias de sombra disponíveis em _tc-rev-shadow.scss.

Exemplos de Sombra

Use as classes abaixo para aplicar diferentes níveis de sombra em elementos. Todas utilizam box-shadow com variações de intensidade:

.shadow-xs
.shadow-sm
.shadow-md
.shadow-lg
.shadow-xl
ClasseDescrição
shadow-xsSombra extra pequena
shadow-smSombra pequena
shadow-mdSombra média
shadow-lgSombra grande
shadow-xlSombra extra grande

Consulte o arquivo styles/scss/_tc-rev-shadow.scss para detalhes técnicos e manutenção.
