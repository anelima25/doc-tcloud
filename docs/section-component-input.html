Input - tc-rev-input-control

O componente tc-rev-input-control oferece estilos padronizados para campos de entrada de texto, busca, radio e checkbox, seguindo o padrão visual do TCloud.

Input padrão


<input class="tc-rev-input-control" type="text" placeholder="Digite algo...">

🔍 
<div class="tc-rev-input-icon-container">
  <input class="tc-rev-input-icon-control" type="text" placeholder="Com ícone à direita">
  <span class="tc-rev-input-icon">🔍</span>
</div>

</div>

Radio button

<input type="radio" class="tc-rev-input-control" name="exemplo-radio" checked>

Checkbox

<input type="checkbox" class="tc-rev-input-checkbox" checked>

Estados

  • Normal: Estado padrão do campo.
  • Hover/Focus: Borda fica azul (primary).
  • Checked: Para radio/checkbox, mostra seleção customizada.
  • Disabled: Use o atributo disabled para desabilitar o campo.
