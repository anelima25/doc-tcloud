Documentação de Bordas

Esta página demonstra as classes utilitárias de borda disponíveis em _tc-rev-border.scss.

Exemplos de Bordas

Use as classes abaixo para aplicar diferentes estilos, larguras e raios de borda em elementos. Os nomes das classes seguem o padrão utilitário para facilitar a aplicação rápida.

.bor-1
.bor-2
.bor-4
.bor-none
.bor-r-16
.bor-r-24
ClasseDescrição
bor-1Borda de 1px
bor-2Borda de 2px
bor-4Borda de 4px
bor-noneSem borda
bor-r-16Borda com raio de 16px
bor-r-24Borda com raio de 24px

Outras classes utilitárias podem estar disponíveis para lados específicos (ex: bor-t-1 para borda superior de 1px) e para diferentes raios de borda.

Consulte o arquivo styles/scss/_tc-rev-border.scss para detalhes técnicos e manutenção.
