Utilitários - tc-rev-util

O arquivo _tc-rev-util.scss reúne classes utilitárias para facilitar a aplicação rápida de estilos comuns em elementos HTML.

Classes disponíveis

  • .cursor-pointer: Aplica cursor: pointer (mãozinha ao passar o mouse).
  • .cursor-default: Aplica cursor: default (seta padrão).
  • .d-block: Aplica display: block.
  • .d-flex: Aplica display: flex.
  • .d-grid: Aplica display: grid.

Exemplos de uso

<button class="cursor-pointer">Botão com cursor pointer</button>
<div class="d-flex">...</div>
<span class="d-block">Texto em bloco</span>

Dicas de aplicação

  • Use .cursor-pointer para indicar elementos clicáveis, como botões e links customizados.
  • .d-flex e .d-grid são úteis para layouts rápidos sem precisar escrever CSS adicional.
  • Combine utilitários para agilizar prototipação e ajustes visuais.
