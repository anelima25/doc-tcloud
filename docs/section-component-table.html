Tabela - tc-rev-table

O componente tc-rev-table oferece uma tabela estilizada, com cabeçalho destacado, alinhamentos customizáveis e separação visual entre linhas.

Exemplo de uso

Nome Idade Salário
Ana 28 R$ 5.000
Bruno 34 R$ 7.200
Carla 25 R$ 4.800 
<table class="tc-rev-table">
  <thead>
    <tr>
      <th>Nome</th>
      <th class="center">Idade</th>
      <th class="right">Salário</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ana</td>
      <td class="center">28</td>
      <td class="right">R$ 5.000</td>
    </tr>
    <tr>
      <td>Bruno</td>
      <td class="center">34</td>
      <td class="right">R$ 7.200</td>
    </tr>
    <tr>
      <td>Carla</td>
      <td class="center">25</td>
      <td class="right">R$ 4.800</td>
    </tr>
  </tbody>
</table>

Alinhamento de colunas

  • .center: Centraliza o conteúdo da célula.
  • .right: Alinha o conteúdo à direita.

Características

  • Cabeçalho com fundo destacado e cantos arredondados.
  • Separação visual entre linhas com borda inferior.
  • Compatível com responsividade (use overflow-x: auto no container se necessário).

Boas práticas

  • Sempre utilize <thead> para o cabeçalho e <tbody> para o corpo da tabela.
  • Use as classes center e right para alinhar colunas conforme necessário.
  • Para tabelas grandes, utilize um container com overflow-x: auto para evitar quebra de layout em telas pequenas.
