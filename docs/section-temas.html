Documentação de Temas

Esta página demonstra o conceito e a aplicação dos temas definidos em _tc-rev-themes.scss.

O que são temas?

Temas permitem alterar rapidamente o visual de componentes ou páginas inteiras, mudando cores, fundos e outros estilos de acordo com o contexto (ex: tema claro, escuro, high-contrast, etc.).

Como aplicar um tema

Normalmente, um tema é aplicado adicionando uma classe específica ao elemento <body> ou a um container principal. As variáveis CSS e estilos definidos para aquele tema passam a valer para todos os elementos filhos.

Exemplo de uso

.theme-light
Exemplo de tema claro
.theme-dark
Exemplo de tema escuro

Você pode definir outros temas, como .theme-high-contrast, e customizar variáveis CSS para cada um.

Exemplo de CSS para temas

:root {
  --background: #fff;
  --text: #222;
}
.theme-dark {
  --background: #222;
  --text: #fff;
}
body, .app-container {
  background: var(--background);
  color: var(--text);
}

Assim, ao adicionar theme-dark ao <body>, todo o site ficará escuro automaticamente.

Consulte o arquivo styles/scss/_tc-rev-themes.scss para detalhes técnicos e manutenção.
