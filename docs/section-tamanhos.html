Tamanhos

Esta página demonstra as classes utilitárias de tamanhos disponíveis em _tc-rev-sizes.scss.

Altura e Largura

Use as classes .h-* e .w-* para definir altura e largura rapidamente. Os valores seguem a escala definida nas variáveis --size-*.

.h-16 .w-16
.h-32 .w-32
.h-48 .w-48
.h-80 .w-80

Padding

Use as classes .pad-* para padding em todos os lados, .pad-x-* para padding horizontal, .pad-y-* para padding vertical, e .pad-l-*, .pad-r-*, .pad-t-*, .pad-b-* para lados específicos.

.pad-16
.pad-x-32
.pad-y-8
.pad-t-24 .pad-b-24

Margin

Use as classes .mar-* para margin em todos os lados, .mar-x-* para margin horizontal, .mar-y-* para margin vertical, e .mar-l-*, .mar-r-*, .mar-t-*, .mar-b-* para lados específicos.

.mar-16
.mar-x-32
.mar-y-8
.mar-t-24 .mar-b-24
