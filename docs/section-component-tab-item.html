Tab Item - tc-rev-tab-item

O componente tc-rev-tab-item é utilizado para criar abas (tabs) estilizadas, facilitando a navegação entre diferentes conteúdos em uma interface.

Exemplo de uso

<button class="tc-rev-tab-item active">Aba 1</button>
<button class="tc-rev-tab-item">Aba 2</button>
<button class="tc-rev-tab-item" disabled>Aba 3</button>

Estados

  • Normal: Aba padrão, cor neutra.
  • Hover/Focus: Fundo e borda ficam mais claros, texto azul.
  • Active/Selected: Fundo azul claro, borda azul, texto azul e negrito.
  • Disabled: Aba desabilitada, cor apagada e cursor não permitido.

Dicas de acessibilidade

  • Use role="tablist" no container das abas e role="tab" nos botões para melhor acessibilidade.
  • Gerencie o foco e o atributo aria-selected para indicar a aba ativa.
<div role="tablist">
  <button class="tc-rev-tab-item" role="tab" aria-selected="true">Aba 1</button>
  <button class="tc-rev-tab-item" role="tab" aria-selected="false">Aba 2</button>
</div>
